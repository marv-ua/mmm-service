Документ "Реестр счетов "  предназначен для печати реестра счетов по 
форме 0401065. 

Для работы с документами "Реестр на инкассо" используется журнал 
"<Прочие>".

Порядок заполнения документа.

Номер документа заполняется автоматически, однако, его можно  редактировать.

Дата документа устанавливается рабочей датей, установленной в настройке 
параметров программы. Дат документа также может быть отредактирована.

Номер, дата и сумма аккредитива. Заполняется вручную - на основании данных 
аккредитива выставленному плательщиком. 

Плательщик - выбрать из предложенного программой справочника "Контрагенты" 
организацию - плательщика по аккредитиву. Если таковая в справочнике отсутствует, 
ее можно ввести в справочник в процессе выписки реестра. Банковские 
реквизиты плательщика подставляются в платежное поручение из справочника 
автоматически.

Счет плательщика - выбором из справочника "Расчетные счета" указать, по каким 
банковским реквизитам плательщика ведутся расчеты по аккредитиву. 
Справочник "Расчетные счета" подчинен справочнику "Контрагенты", поэтому в 
открывающемся окне выбора показываются только те расчетные счета, которые 
относятся к конкретному плательщику.

Срок закрытия  аккредитива - указывается дата закрытия аккредитива (в формате 
ДД.ММ.ГГГГ).

Комментарий - необязательный реквизит, не выводится на печать и служит лишь для 
записи информации, связанной с документом.         

По кнопке "Печать" производится вывод реестра на печать.
Согласно п. 6.4 части 1 Положения ЦБ РФ от 12.04.2001г. № 2-П, реестр 
должен составляться в четырех экземплярах. 

В табличной части документа указываются данные по отгрузочным
и другим документам, предусмотренным условиями аккредитива.
