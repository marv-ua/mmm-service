Отчет "Список номенклатуры" предназначен для печати справочника номенклатуры и
информации по каждому из его элементов (себестоимость, ставки налогов и пр.) 

Отчет вызывается непосредственно из формы списка справочника "Номенклатура"
с помощью кнопки "Печать списка" или  из меню "Отчеты".

Если при настройке параметров отчета указать конкретную группу справочника,
то список будет составлен только из элементов справочника, принадлежащих выбранной группе.
Если группу не выбирать, то список будет формироваться из всех элементов справочника.

В отчете предусмотрена возможность установки отбора элементов, по конкретному
значению любого реквизита справочника. Элементы справочника, со значением
выбранного реквизита отличным от указанного значения, в отчет включаться не будут. 
Например, задав в отборе параметры  "Тип номенклатуры" и "Товар", мы получим
список номенклатуры, состоящий только из товаров.  

Если отчет вызывается из справочника, то фильтр автоматически принимает значение
установленного в справочнике отбора.
