	Обработка предназначена для формирования остатков на вспомогательных 
забалансовых счетах налогового учета по данным бухгалтерского учета.

	Элементы диалога:
-------------------------------------------------------------------------------
- 	"Остатки на дату": дата начала учета, на которую выводятся начальные 
	остатки. Проводки по вводу начальных остатков формируются предыдущим днем. 
- 	"Раздел учета": вспомогательные забалансовые счета налогового учета, 
	соотвествующие разделу налогового учета.
- 	"Обновить": обновляются данные бухгалтерского и налогового учета для 
	отражения в отчете.
- 	"Сформировать проводки": по данным бухгалтерского учета, формируются 
	входящие остатки. Бухгалтерская операция по вводу начальных остатков
	формируются датой, предшествующей дате, указанной в поле "Остатки на дату".
	Существующие операции помечаются на удаление.

	Графы таблицы:
-------------------------------------------------------------------------------
- 	"Данные по бухг. учету": расчетные данные для формирования остатков по 
	налоговому учету.
- 	"Данные по налоговому учету": текущие начальные остатки по налоговому 
	учету на указанную дату.  Двойной щелчок мыши или нажатие клавиши "Enter" на
	ячейке вызывает отчет по проводкам, сформировавшим  остатки.
- 	"Отклонение": разница между текущими остатками по налоговому учету и 
	расчетеыми по бухгалтерскому учету.

	Для счета Н05.01, Н05.03 выводятся дополнительные графы:
-------------------------------------------------------------------------------
- 	"Исходная амортизация по бухг. учету": расчетное значение амортизации, 
	предполагаемое к принятию в налоговом учете. Формируется только при 
	установке в графе таблицы "Способ принятия начальных остатков к налоговому 
	учету" значения  "по балансовой стоимости".
- 	"Начисленная амортизация по налоговому учету": текущее значение 
	амортизации объекта, принятое к налоговому учету.  Двойной щелчок мыши или 
	нажатие клавиши "Enter" на ячейке вызывает отчет по проводкам, 
	сформировавшим остатки.
-	"Первоначальная стоимость по НУ", "Срок полезного использования (по НУ), мес", 
	"Амортизационная группа", "Метод начисления амортизации", "Специальный 
	коэффициент" - реквизиты справочника "Основные средства".  Двойной щелчок мыши
	или нажатие клавиши "Enter" на ячейке открывает соответствующий элемент 
	справочника на закладке "Налоговый учет".
-	"Способ принятия начальных остатков к налоговому учету": может принимать 
	значения "по балансовой стоимости" или "по остаточной стоимости". Переключение
	производится при двойном щелчке мыши или нажатии клавиши "Enter" на ячейке. 
	
	Для счета Н13.02 выводятся дополнительные графы: 
-------------------------------------------------------------------------------
-	"НДС по кред. задолженности по бухг. учету": расчетное значение НДС по 
	кредиторской задолженности, предполагаемое к принятию в налоговом учете.
-	"НДС по кред. задолженности по налоговому учету": текущее значение НДС по 
	кредиторской задолженности, принятое к налоговому учету.  Двойной щелчок мыши 
	или	нажатие клавиши "Enter" на ячейке вызывает отчет по проводкам, 
	сформировавшим  остатки. 

	Разделы учета
-------------------------------------------------------------------------------
-	Объекты внеоборотных активов
	-	Данные бухгалтерского учета: СКД(08)
	- 	Данные налогового учета: СКД(Н01.01)

-	Оборудование
	- 	Данные бухгалтерского учета: СКД(07)
	-	Данные налогового учета: СКД(Н01.02)

-	Учет прямых расходов на производство товаров (работ, услуг) основного производства
	-	Данные бухгалтерского учета: СКД(20)
	-	Данные налогового учета: СКД(Н01.05)
	-	Группы (виды) расходов берутся из реквизита "Вид расходов" справочника 
		"Статьи затрат на производство"

-	Транспортные расходы по доставке товаров
	-	Данные бухгалтерского учета: СКД(44.1.1)
	-	Данные налогового учета: СКД(Н01.07)
	-	Включаются элементы справочника "Статьи издержек обращения", списывающиеся 
		пропорционально остатку товаров

-	Материалы
	-	Данные бухгалтерского учета: СКД(10)
	-	Данные налогового учета: СКД(Н02.01)

-	Товары
	-	Данные бухгалтерского учета: СКД(41)
	-	Данные налогового учета: СКД(Н02.02)

-	Готовая продукция
	-	Данные бухгалтерского учета: СКД(43)
	-	Данные налогового учета: СКД(Н02.03)

-	Товары отгруженные
	-	Данные бухгалтерского учета: СКД(45)
	-	Данные налогового учета: СКД(Н02.04)

-	Ценные бумаги
	-	Данные бухгалтерского учета: СКД(58.2)
	-	Данные налогового учета: СКД(Н02.05)

-	Права в рамках оказания финансовых услуг
	-	Данные бухгалтерского учета: СКД(58.5)
	-	Данные налогового учета: СКД(Н02.05)
	
-	Расходы будущих периодов
	-	Данные бухгалтерского учета: СКД(97)
	-	Данные налогового учета: СКД(Н04.09)
	
-	Основные средства
	-	Балансовая стоимость по БУ (СКД(01, 03)). При способе принятия начальных 
		остатков к налоговому учету по балансовой стоимости отражается балансовая
		стоимость, по остаточной стоимости - остаточная. Если остаточная стоимость
		меньше 10000 руб, то нулевая стоимость.
	-	Амортизация по БУ (СКК(02))
	-	Первоначальная (восстановительная) стоимость по НУ (СКД(Н05.01))
	-	Амортизация по НУ (СКК(Н05.02))

-	Нематериальные активы
	-	Балансовая стоимость по БУ (СКД(04) или реквизит "Первоначальная стоимость")
		При способе принятия начальных остатков к налоговому учету по балансовой 
		стоимости отражается балансовая стоимость, по остаточной стоимости - 
		остаточная. Если остаточная стоимость меньше 10000 руб, то нулевая стоимость.
	-	Амортизация по БУ (СКК(05) или разница между 04 или реквизитом 
		"Первоначальная стоимость")
	-	Первоначальная стоимость по НУ (СКД(Н05.03))
	-	Амортизация по НУ (СКК(Н05.04))

-	Дебиторская задолженность
	-	Данные бухгалтерского учета: дебетовое сальдо по счетам 58.3, 60.2, 60.22, 62.1, 62.11, 76.5.
	-	Данные налогового учета: дебетовое сальдо по счету Н13.01

-	Кредиторская задолженность
	-	Данные бухгалтерского учета: кредитовое сальдо по счетам 60.1, 60.11, 62.2, 62.22, 66, 67, 76.5.
	-	Данные налогового учета: кредитовое сальдо по счету Н13.02
	- 	НДС по кредиторской задолженности:
		-	по счету 62.2: на основании счета 76.АВ;
		- 	по счету 60.1: на основании счета 19.
