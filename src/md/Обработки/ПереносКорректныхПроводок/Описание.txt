Обработка "Перенос корректных проводок" предназначена для передачи
списка корректных проводок между информационными базами.

Обработка дает возможность выгрузить список корректных проводок
в текстовый файл и загрузить их из файла. Файл переноса может быть
просмотрен в виде таблицы и, при необходимости распечатан.

Загрузка корректных проводок предусматривает два режима:
- Добавить только новые проводки
- Загрузить все (существующие удалить)
В первом режиме из файла переноса в список корректных проводок попадают
только ранее отсутствовавшие корреспонденции.
Во втором режиме существующий список заменяется на загружаемый из файла.
