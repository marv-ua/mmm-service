Запуск обработки выполняется из системы "1С:Предприятие  7.7" (при любом составе установленных компонент). 
Для запуска обработки откройте файл UChoice.ert с помощью пункта "Открыть" из меню "Файл" главного меню системы
1С:Предприятие или настройте вызов из произвольной формы.

Вызов обработки производится с помощью команды ОткрытьФорму() или ОткрытьПодбор(), в которой в качестве
параметра <КонтекстФормы> выступает список значений со следующим возможным набором значений
(здесь указывается строковое представление значений).

Параметры вызова 

Все параметры являются необязательными.

Общие

·	ИмяВызвавшейФормы		- Строка;  "Отчет." + ИмяОтчета; или "Обработка." + ИмяОтчета и т.п.; или ПолноеИмяВнешнегоФайла (.ert)
·	ДатаНачалаИнтервала		- Дата; для выбора и фильтрации документов (операций)
·	ДатаКонцаИнтервала		- Дата; для выбора и фильтрации документов (операций)
·	ИспользоватьЗакладки	- "Нет"; если нужен упрощенный режим - только интерактивного подбора, без фильтров
·	ЗапоминатьНомераСтрок	- Число 1, если необходимо запоминать номера срок документа или номера проводок, 
	удовлетворяющих условиям (например, найти проводки со счетом "КодСчета", или найти строки с товаром "Товар").
	Запомненные номера строк/проводок будут записаны через запятую в строковое представление возвращаемого списка
	значений "Объекты" ("1,3,8,16,26" и т.д.) - пример использования см. Plug-In UCSetAtt (обработка установки реквизитов)

Для интерактивной настройки отбора

·	Тип				- Строка; Тип объекта, по которому производится интерактивный отбор
·	Вид				- Строка; Вид объекта, по которому производится интерактивный отбор
·	Действия		- СписокЗначений;  некоторый список для интерактивного выбора значения; выбранное значение будет возвращено.
·	Объекты			- СписокЗначений; для начального заполнения; передается по ссылке, т.е. после окончания, сюда вернется список
					отобранных объектов (это удобно когда список значений встроен в форму и нужно его заполнить например по кнопке <?>).
					Для документов возможен также вариант "ВсеВИнтервале", означающий , что главную таблицу  обработки перед началом
					работы необходимо заполнить всеми документами из интервала "ДатаНачалаИнтервала"  - "ДатаКонцаИнтервала".
·	Буфер			- Любое значение; сквозная переменная - будет  возвращена в вызывающую форму. C помощью данного параметра в
					процедуре ПриПовторномОткрытии() вызывающей формы, возможно отследить откуда и в какой момент была вызвана
					данная обработка подбора объектов
·	Владелец		- Элемент справочника владельца, если нужно отбирать элементы подчиненного справочника с включенной иерархией
·	БезГрупп		- Число 1, если нужно недопусить отбор групп справочника
·	НепоказываемыеРеквизиты	- СписокЗначений; рекизиты объектов, которые не следует показывать в данной обработке. 

Например: для Торговли 8.7 это

НепоказываемыеРеквизиты.ДобавитьЗначение("РозничнаяЦена",			"Номенклатура");
НепоказываемыеРеквизиты.ДобавитьЗначение("МелкооптоваяЦена",		"Номенклатура");
НепоказываемыеРеквизиты.ДобавитьЗначение("ОптоваяЦена",				"Номенклатура");
НепоказываемыеРеквизиты.ДобавитьЗначение("ЦенаПриобретения",		"Номенклатура"); 

НепоказываемыеРеквизиты.ДобавитьЗначение("ВесьСправочник",			"КатегорииТоваров");

НепоказываемыеРеквизиты.ДобавитьЗначение("Код",						"СвойстваНоменклатуры");
НепоказываемыеРеквизиты.ДобавитьЗначение("Наименование",			"СвойстваНоменклатуры");

И т.д.

Параметры возврата

·	Объекты					- СписокЗначений; отобранных объектов
·	ВыбранноеДействие		- Строка; возвращается выбранное значение из переданного при вызове "СпискаДействий"
·	Буфер					- Любое значение; назначение описано выше
