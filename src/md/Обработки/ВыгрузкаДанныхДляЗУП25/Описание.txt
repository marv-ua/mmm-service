Обработка "Универсальная выгрузка данных в формате XML".

Предназначена для выгрузки данных в файл из любой конфигурации,
реализованной на платформе 1С:Предприятие версии 7.7.

ВАЖНО!!!
Данная обработка предназначена для выгрузки данных по правилам, настроенным
в конфигурации "Конвертация данных, редакция 2", версия 2.0.11.1.

Выгрузка данных.
Перед осуществлением выгрузки необходимо открыть данную обработку в "1С:Предприятие 7.7"
в режиме конфигуратора и полность заменить модуль данной обработки текстом, полученным
при сохраненнии правил с включенным флагом " Выгружать модуль обрарботки выгрузки". В этом случае модуль будет
содержать все необходимые обработчики событий и алгоритмы, соответствующие правилам конвертации,
настроенным в конфигурации "Конвертация данных, редакция 2.0", версия 2.0.11.1. Данную процедуру необходимо
выполнять после каждого изменения какого либо из обработчиков или алгоритмовв правилах обмена.
Подробное описание процедуры создания модуля обработки приведено в справочной информации
конфигурации "Конвертация данных, редакция 2".

Для осуществления выгрузки данных необходимо указать имя файла, в который будет осуществляться
выгрузка данных и выбрать файл правил обмена. Правила обмена для любых конфигураций могут быть
настроены в специализированной конфигурации "Конвертация данных, редакция 2.0". Далее необходимо
загрузить правила обмена, нажав кнопку "Загрузить правила обмена". Для выгрузки документов за
определенный интервал необходимо указать период - "Дату начала" и "Дату окончания". Флажки в списке
"Правила выгрузки данных" указывают, какие данные будут выгружаться, а какие нет.
