Обработка "Универсальная загрузка данных в формате XML".

Предназначена для загрузки данных в файл в любую конфигурацию,
реализованную на платформе 1С:Предприятие версии 7.7.

ВАЖНО!!!
Данная обработка предназначена для загрузки данных по правилам, настроенным
в конфигурации "Конвертация данных, редакция 2", версия 2.0.10.1.

Выгрузка данных.
Перед осуществлением загрузки необходимо открыть данную обработку в "1С:Предприятие 7.7"
в режиме конфигуратора и полность заменить модуль данной обработки текстом, полученным
при сохраненнии правил с включенным флагом " Выгружать модуль". В этом случае модуль будет
содержать все необходимые обработчики событий и алгоритмы, соответствующие правилам конвертации,
настроенным в конфигурации "Конвертация данных, редакция 2", версия 2.0.10.1. Данную процедуру необходимо
выполнять после каждого изменения какого либо из обработчиков или алгоритмовв правилах обмена.
Подробное описание процедуры создания модуля обработки приведено в справочной информации
конфигурации "Конвертация данных, редакция 2".

Для осуществления загрузки данных необходимо указать имя файла, из которого будет осуществляться
загрузка данных. Правила обмена для любых конфигураций могут быть настроены в специализированной 
конфигурации "Конвертация данных, редакция 2.0".
