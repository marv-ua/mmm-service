Обработка "Выбор из классификатора" предназначена для просмотра 
"Классификатора банков РФ", "Классификатора валют" и "Классификатора единиц измерения"
и для добавления выбранных объекток в соответствующий справочник.

Обработка вызывается в режиме подбора (позволяет добавить сразу несколько несколько) 
из "Основной" формы справочника, и в режиме выбора (позволяет добавить один выбранный 
объект) из формы "Для выбора".

Классификатор банков хранится во внешнем файле "Bik.dbf". 
Классификатор валют хранится во внешнем файле "Val.dbf". 
Классификатор единиц измерения хранится во внешнем файле "OKEI.dbf". 

Список банков двухуровневый: на первом уровне содержатся регионы РФ, на втором - банки. 
Для открытия нужного региона надо дважды щелкнуть на нем мышью или нажать "Enter". 
Открытый регион будет отображен в первой строке списка. Для возвращения на первый 
уровень необходимо дважды щелкнуть на нем мышью или нажать "Enter".

Список валют одноуровневый. 

Список единиц измерения двухуровневый: на первом уровне содержатся группы единиц измерения, 
на втором - единицы измерения. Для открытия нужной группы надо дважды щелкнуть на ней мышью 
или нажать "Enter".  Открытая группа будет отображена в первой строке списка. Для возвращения 
на первый  уровень необходимо дважды щелкнуть на ней мышью или нажать "Enter".

Классификатор можно отсортировать одним из предложенных способов.

Для выбора объекта необходимо дважды щелкнуть на нем мышью или нажать "Enter", 
или выбрать его в списке и нажать кнопку "Выбрать". Если объект отсутствовал в 
справочнике, он будет добавлен. Если обработка открыта для выбора одного значения, - 
она закроется; если обработка открыта для подбора - выбор объектов может быть продолжен.

Классификатор банков, валют и единиц измерения можно регулярно обновлять с диска 
"Информационно-технологическое сопровождение "1С:Предприятие""  (начиная с ??? 2001 года), 
кроме того  классификатор банков может обновлятся с web-сайта агентства 
"РосБизнесКонсалтинг" (www.rbc.ru). 

Для обновления классификатора следует нажать кнопку "Обновить". При нажатии на нее 
вызывается обработка "Обновление классификатора". Если классификатор пустой, то эта обработка 
будет вызвана автоматически.
