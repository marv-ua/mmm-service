Справочник "Грузовые таможенные декларации" (ГТД) содержит список номеров ГТД,
по которым импортные товары ввезены в страну. Для каждого номера ГТД в справочнике 
предусмотрено поле комментария. При формировании счета-фактуры номер ГТД подставляется 
в документ как реквизит табличной части.

Справочник "Грузовые таможенные декларации" используется также для ведения аналитического
учета на забалансовом счете ГТД "Учет импортных товаров по ГТД".
