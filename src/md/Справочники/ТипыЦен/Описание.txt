    Справочник «Типы цен»
Количество продажных цен на ТМЦ , которые можно задать для одной
номенклатурной единицы, в конфигурации не ограничено. Для этого
в справочник вводится несколько элементов, соответствующих
используемым компанией типам цен (например: «розничная», «
оптовая», «дилерская» и т.д.). Для каждого склада может быть
задан свой тип цен. Для каждой позиции номенклатуры может быть
назначено столько цен, сколько их изначально определено в
справочнике «Типы цен».
Добавление и редактирование информации производится в
специальном диалоговом окне.

В реквизите «Наименование» указывается название типа цен,
например, «Оптовая (со склада готовой продукции)».
Значения установленные в группе «Цены:» используются для
подстановки по умолчанию при создании новой цены данного типа.
