Справочник "Настройка раскрытия отчетности" содержит разделы бухгалтерской
отчетности и конкретные объекты учета, по которым раскрываются отдельные 
показатели бухгалтерской отчетности (в формах NN 1 и 2).

Справочник заполняется пользователем по мере необходимости раскрытия
отдельных показателей  по тем или иным разделам отчетности и объектам
учета.

На верхнем уровне справочника указываются разделы отчетности. 
Для указания раздела следует выбрать меню "Действия - Новая группа",
в колонке "Объект учета" выбрать из списка требуемый раздел и задать
в колонке "Уровень" уровень существенности для выбранного раздела.
                                                                                                                              
Для некоторых разделов отчетности, в частности,  для основных средств
и нематериальных активов, возможна расшифровка по видам объектов 
учета.  Для задания расшифровки по видам объектов учета необходимо
предварительно открыть соответствующий раздел учета (меню 
"Действия - Следующий уровень"). Порядок ввода  нового вида объектов
аналогичен порядку ввода нового раздела учета. 

Следует отметить, что уровень существенности, заданный для вида
объектов, распространяется на все объекты учета данного вида.
Уровень существенности для вида объектов может отличаться
от уровня существенности соответствующего раздела отчетности.

При необходимости раскрывать данные по конкретному объекту учета,
независимо от заданного уровня существенности, пользователь должен
включать такие объекты в настройку. Для задания обязательной расшифровки
по конкретному объекту следует предварительно открыть соответствующий
раздел (или вид объектов при наличии),  выбрать меню "Действия - Новый"
и в колонке "Объект учета" выбрать из списка требуемый объект учета.
